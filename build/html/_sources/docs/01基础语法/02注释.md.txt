# 注释

### 1.2 注释
1. 单行注释和多行注释；
2. 块注释是不能嵌套的，会提示输入表达式而非；

```c++
// 这是单行注释
/*
	这是多行
	注释
*/
```

**块注释嵌套**
```c++
int main() {
	// 会提示输入表达式而非*/
	/* int v1 = 0, v2 = 0;
	 * std::cout << "The sum of " << v1
	 * << " and " << v2
	 * << " is " << v1 + v2 << std::endl;/**/
	 */
}
```

**判断是否合法**
```c++
std::cout << "/*" << std::endl; //成立，会输出/*
std::cout << "*/" << std::endl; //成立，会输出*/
//std::cout << /* "*/" */; //报错
std::cout << /* "*/" /* "/*" */ << std::endl; //成立，会输出/*
```
