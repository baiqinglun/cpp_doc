常量

  1. “#define定义宏常量”：#define” 常量名 常量值` (通常在文件上方定义，表示一个常量)；

  2. const修饰的变量 const 数据类型 常量名 = 变量值;

  3. 注意#define语句后面不加分号；

#include <iostream>
using namespace std;

// 常量定义方式
// 1.#define
// 2.const修饰的变量

#define Day 7

int main() {
	cout << "一周共有"<< Day << "天" << endl;

	const int month = 12; // const修饰的变量也是常量，不能修改
	system("pause");
	return 0;

}
